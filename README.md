# Healthcare Data Analysis Dashboard (Power BI)

# Overview
This project analyzes patient demographic and healthcare data using Power BI.
The goal is to uncover insights related to:

Demographics (age, gender, ethnicity),
Geographic distribution,
Mortality patterns,
Population trends and migration.
 
 Dashboard Structure
1. Demographics
2. Geographic Insights
3. Mortality Analysis
4. Trends & Migration


# Key Questions Answered
This project answers 21 analytical questions, including:
1.	Are certain demographic groups (age, gender, race) more concentrated in specific cities or counties?
2.	Do older patient groups tend to cluster in particular ZIP codes?
3.	Is there a noticeable shift in patient age distribution across different birth decades?
4.	Does marital status appear to correlate with certain age ranges or regions?
5.	Are unmarried patients more common in specific neighborhoods?
6.	Do certain cities show higher diversity in racial or ethnic groups?
7.	Is mortality higher among patients born in certain years or decades?
8.	Do deceased patients tend to come from particular counties more than others?
9.	Does age at death vary significantly between different ethnic groups?
10.	Are there geographic hotspots with unusually high or low patient density?
11.	Do rural vs urban ZIP codes show differences in demographic structure?
12.	Is there a geographic pattern in patient longevity or death occurrence?
13.	Do multiple patients share similar address patterns, indicating household clusters?
14.	Are certain demographic groups more likely to live in high-density ZIP codes?
15.	Is there any relationship between birthplace and current living city?
16.	Are younger patients gradually increasing in a particular city, indicating demographic shifts?
17.	Do birthplaces indicate migration trends into certain counties?
18.	Is there a pattern in ethnicity distribution changing across age groups?
19.	Are there ZIP codes where the gender ratio is significantly imbalanced?
20.	Are there unusually high-age or low-age clusters in some areas?
21.	Do some counties show unexpectedly low ethnic diversity?
